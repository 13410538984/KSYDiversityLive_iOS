# 金山云-商汤科技动态贴纸（AR直播）

##1.概述

金山视频云拥有功能全面的推流和拉流解决方案，商汤在图像识别和图像处理有多年的技术积累，两家在各自领域的强者结合一定会产生出不一样的效果，下面我们就介绍一下，集成了金山采集、编码、推流功能和商汤人脸识别、图像处理功能的例子。
